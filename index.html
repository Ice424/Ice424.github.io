Origins 5E

Origins 5E is a project involving Minecraft's resource and datapack features along with the features intrude in the origins mod. It uses python to take powers defined in the origins format & automatically add them to a custom GUI system built on this project which allows players to buy and equip powers with a new currency found across the world.

This project contains over 100 powers and over 130 pixel art assets for GUI screens, power icons, etc. Due to this this has lead me to teach others on how to use git & github and the python libraries that I have created to be able to aid me with this project
